Всем привет!
Представляю Вам моё приложение по заказу пиццы!
Цель разработки данного приложения - отработка принципов навигации между экранами.
На первом экране приложения можно выбрать интересующий вкус пиццы, на втором, третьем, чертвёртом - размер пиццы, необходимое количество, дату доставки. Пятый экран - результирующий экран заказа.
Так же создан свой логотип :) Pizza Joe - Пиццерия Георгия :)
